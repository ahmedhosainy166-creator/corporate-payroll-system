# نظام إدارة الموارد البشرية ورواتب الشركات المتكامل

برنامج مصمم بلغة بايثون لأتمتة عمليات الموارد البشرية (HR) وحساب الرواتب، والمكافآت، والاستقطاعات الضريبية تصاعدياً بناءً على تقييم الأداء وصافي الدخل للموظفين.

## 🚀 مميزات النظام
* **حساب المكافآت تلقائياً:** يمنح مكافآت بنسبة 20% أو 10% بناءً على تقييم أداء الموظف (من 1 إلى 5).
* **خصم الضرائب التصاعدية:** يطبق نسبة ضريبة 10% أو 15% تلقائياً بناءً على إجمالي دخل الموظف لتفادي الأخطاء البشرية.
* **التحقق من صحة البيانات (Input Validation):** يمنع الأخطاء المنطقية (مثل إدخال راتب بالسالب أو تقييم خارج النطاق) ويقوم بإيقاف البرنامج لحماية الحسابات.
* **إصدار تقارير مالية منسقة:** يطبع قسيمة راتب (Payroll Statement) احترافية ومنظمة مع تقريب مالي دقيق لأقرب قرشين باستخدام تنسيق الأرقام العشري (`:.2f`).

## 🛠️ المفاهيم البرمجية المستخدمة
* **اللغة:** Python 3
* **المفاهيم:** الدوال المعزولة (Functions)، المنطق الشرطي المتعدد (`if-elif-else`)، معالجة المدخلات، وتنسيق النصوص المتقدم (f-strings).
